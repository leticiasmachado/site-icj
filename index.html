Caracterização de Cintilações em Receptores GPS de Navegação – Pesquisa de Iniciação Científica Júnior

Sobre a pesquisa

A pesquisa tem apoio do INCT (Instituto Nacional de Ciência e Tecnologia) e está inserida no eixo de melhorias no desempenho de receptores GNSS sob a incidência de cintilações ionosféricas e interferências. A pesquisa deve analisar dados de monitoramento de receptores com o objetivo de se compreender melhor as circunstâncias na qual um receptor pode perder o rastreamento de um sinal GPS devido à cintilação forte. Utiliza-se o ISMR Query Tool, software científico para obtenção dos dados de monitoramento de cintilações e dados de receptores GNSS.

Os resultados que foram obtidos até então permitirão a obtenção de subsídios para futuras implementações na arquitetura de receptores GNSS, a fim de que os mesmos se tornem mais resistentes a possíveis incidências de cintilações. Também podemos usar os resultados para a emissão de alertas e buscar diminuir efeitos de cintilação nos receptores.

Glossário

GNSS: sistemas de posicionamento por satélites artificiais com cobertura mundial.

Cintilações: flutuações na intensidade do sinal. Ocorrem na ionosfera terrestre e causam um sinal inconstante.

Índice S4: índice médio de desvio do sinal rastreamento.

ISMR Query Tool: software científico para obtenção dos dados de monitoramento de cintilações. Está disponível em um site mantido pela UNESP. Através deles conseguimos analisar os índices de cintilação e os dados de alta taxa de um receptor GNSS. Para acessá-lo, clique aqui.

Dados de alta taxa: dados brutos captados pelo satélite que possuem taxa de amostragem de 50Hz .

Índices de cintilação: são um resumo dos dados de alta taxa e possuem taxa de amostragem de 60 segundos.

Resultados obtidos

Inicialmente, realizou-se uma pesquisa acerca do assunto abordado, para compreender melhor a metodologia utilizada em receptores GNSS e em satélites, focando também em aprofundar conhecimentos sobre os termos descritos no glossário. A ferramenta utilizada para o download dos dados de satélites foi o ISMR Query Tool, que faz monitoramento de diversos satélites da América do Sul. O foco foi na estação de monitoramento de São José dos Campos (SJCU) no ano de 2014, pelo fato de que durante o ano houve monitoramento em quase todos os dias, tendo assim uma grande quantidade de dados para análise. Algumas das ferramentas disponíveis no site do ISMR Query Tool são a visão de calendário (por ano e por estação) e o plot, para fazer gráficos de nível de cintilação em determinado momento em determinada estação.

Durante a segunda etapa, foi desenvolvido um código no RStudio para melhor analisar momentos de perda de sinal. Através dos plots dos gráficos é possível ver claramente um momento de interrupção no sinal.

Outros exemplos de gráficos desenvolvidos podem ser analisados a seguir.

Atualmente, estamos desenvolvendo um programa em Java que seja capaz de fazer o download automático dos dados dos satélites, bem como analisá-los a fim de apresentar um relatório sobre a interrupção na propagação do sinal, informando quantas foram as perdas, em que momento, qual foi a duração e a intensidade do sinal no momento em que elas ocorreram. Quando concluído, esse software poderá ser usado para pesquisas na área, uma vez que inserindo os dados a serem observados, teremos facilmente um relatório completo dos mesmos.
